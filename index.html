CodeMirror 是一款使用JavaScript实现，面向浏览器的多功能文本编辑器。 专注于代码编辑, 支持大部分的 语言模式 和 插件扩展 是可以让您实现更高级编辑器的最佳基础库.

丰富的 API 和CSS 主题系统 ，通过定制CodeMirror，就能满足您需要的应用程序，扩展它新的功能.

特征

  • 编辑器支持 超过 100 种语言。
  • 强大的混合多语言 模式库
  • 自动完成 (XML)
  • 代码折叠
  • 自定义键盘快捷键
  • Vim, Emacs, 和 Sublime Text 编辑器模式
  • 查找与替换 功能
  • 括号 和 标签 匹配支持
  • 支持 分割视图
  • Linter集成
  • 混合字体的大小和颜色
  • 丰富的主题风格
  • 可以自动 根据内容改变编辑器的高度
  • Inline 和 block 插件
  • 可编程的gutters
  • 为一定范围的文本 添加样式、设为只读、或者原样
  • 双向文本 支持
  • 更多其它的 功能 及 扩展插件...

项目组织

CodeMirror 是一个开源项目，基于MIT license协议. 适合于编辑器的二次开发。 Firefox 、 Chrome、Light Table、 Adobe Brackets、 Bitbucket, 以及 更多项目.

开发和错误跟踪 github (alternate git repository). 请 阅读指引 before submitting a bug. Use pull requests to submit patches. All contributions must be released under the same MIT license that CodeMirror uses.

项目讨论社区 discussion forum. 还有这个 codemirror-announce list, 仅用于重大公告 (例如版本更新). 如果有需要, 您可以发邮件给我们 marijnh@gmail.com directly. We aim to be an inclusive, welcoming community. To make that explicit, we have a code of conduct that applies to communication around the project.

A list of CodeMirror-related software that is not part of the main distribution is maintained on our wiki. Feel free to add your project.
